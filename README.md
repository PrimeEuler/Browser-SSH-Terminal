SSH-Browser-Terminal
====================

A trivial example of a SSH terminal in your browser using node.js, express.io and ssh2. Based on Fabrice
Bellard's vt100 for [jslinux](http://bellard.org/jslinux/).

Install:

```bash
npm install express.io
npm install ssh2
```
Run:
```bash
node server.js
```
Connect:
```bash
http://localhost:8080
```
